Test Case PERF-0020
Title Pulse width resolution is 1 microsecond
Objective

This test case is intended to verify the following implied or specified requirements:

  • servod shall be able to set pulse widths with at least 1 usec resolution.

This test case verifies that the user has the ability to adjust the pulse width with 1 microsecond precision.
Setup
Actions
  1. Start the servod daemon with the repository servoDefinitionFile.txt.
  2. Start pigpiod with 1 usec resolution.
  3. Set each servo to 1500 usec + the servo number (e.g. echo "5, 1505" >/dev/servo_fifo).
  4. Using piscope, verify that each pulse width is unique and equal to the setting.
Expected Pulse widths of the servo commands will range from 1500 to 1509 with resolution of 1 usec.
Results PASSED
Cleanup
