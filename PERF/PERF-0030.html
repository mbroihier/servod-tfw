Test Case PERF-0030
Title CPU load is less than 1%
Objective

This test case is intended to verify the following implied or specified requirements:

  • servod shall require less that 1% CPU utilization after initialization.

This test case demonstrates that the CPU load is small (less than 1%).
Setup
Actions
  1. Using the repository servoDefinitionFile.txt, run servod as a daemon. \bullet sudo ./servod
  2. Using a bash while loop, write a servo command to all ten servos and then wait a second.
  3. While the loop is running, from another terminal run htop and record the CPU utilization of servod.
  4. Verify the utilization is less than 1%.
Expected
Results PASSED
Cleanup
Requirements SERVO-PERF-030
