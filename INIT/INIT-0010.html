Test Case INIT-0010
Title Daemon starts at boot
Objective

This test case is intended to verify the following implied or specified requirements:

  • On boot, the servod shall read its configuration and set the servos to their initial state.
  • The initial state of servod shall continuously send initial pulse widths to all the configured servos.
  • servod shall initialize a FIFO pipe at location /dev/servo_fifo.
  • Servo control shall be through a FIFO pipe.
  • Servo output shall be 3.3 volt PWM signals to configurable GPIO pins.

This test case verifies that the daemon starts at boot and initializes the servos to a known state.
Setup
Actions
  1. Install servod from the repository.
  2. Using the README instructions, compile servod.
  3. Using the README instructions, install servod as a service and enable it.
  4. Boot the Pi.
  5. Using piscope, verify that all the configured servos are being sent pulses of width 1.5 msec.
  6. Using ls -l, verify that the FIFO, /dev/servo_fifo, exists.
Expected servod starts on boot, initializes the configured servos to be at their null position (50% of the range), and opens a FIFO named /dev/servo_fifo
Results PASSED
Cleanup
Requirements SERVO-PR-BOOT-010, SERVO-PR-BOOT-020, SERVO-PR-BOOT-030, SERVO-EXT-IF-010, SERVO-EXT-IF-020
